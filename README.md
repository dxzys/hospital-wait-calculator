# hospital-wait-calculator

A tool that takes a New Brunswick postal code and finds the nearest hospital emergency departments, ranked by distance

## Data

* [Emergency Department Visit Duration](https://experience.arcgis.com/experience/21e1ed7c4ee445d88511757555b8aef2/page/Emergency-Department-Visit-Duration)
* [geocoder.ca](https://geocoder.ca)
* [Open Source Routing Machine](https://project-osrm.org/)
